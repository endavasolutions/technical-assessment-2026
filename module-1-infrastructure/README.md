# Module 1: Infrastructure & Operations

## Objective
Demonstrate ability to read technical documentation, interpret system configurations, and identify operational improvements.

## Scenario
You are reviewing the infrastructure documentation for a 750-employee organization with three office locations. The current setup includes on-premise servers, network equipment, and cloud services.

## Tasks

### Task 1: Documentation Review
Review the provided system architecture document in `documentation/architecture_overview.pdf`. Identify:
- Single points of failure
- Capacity constraints
- End-of-life components
- Missing redundancy

### Task 2: Configuration Assessment
Based on the documentation, answer the following:
1. What is the current backup strategy and what are its weaknesses?
2. How would you improve disaster recovery?
3. What monitoring gaps exist?

### Task 3: Executive Summary
Write a one-page executive summary of your findings with prioritized recommendations.

## Deliverables
- Completed assessment worksheet
- Executive summary document

## Evaluation Criteria
- Accuracy of technical assessment
- Quality of recommendations
- Clarity of executive communication
